# 🖥️ دليل التثبيت السريع على Windows

## 🎯 التثبيت بنقرة واحدة

### الطريقة الأسهل (موصى بها)
1. **حمل ملف التثبيت**
   - `ONE_CLICK_DEPLOY.bat`

2. **شغل الملف**
   - انقر نقرة مزدوجة على الملف
   - اختر "Yes" عند طلب صلاحيات المدير
   - انتظر 5-10 دقائق حتى يكتمل التثبيت

3. **النتيجة**
   - النظام سيفتح تلقائياً في المتصفح
   - اختصارات على سطح المكتب للإدارة

## 📋 متطلبات النظام

### الحد الأدنى
- Windows 10 أو أحدث
- 4GB ذاكرة RAM
- 10GB مساحة تخزين
- اتصال إنترنت

### موصى به
- Windows 11 أو Windows Server 2022
- 8GB ذاكرة RAM
- 20GB مساحة تخزين
- اتصال إنترنت سريع

## 🖱️ اختصارات سطح المكتب

بعد التثبيت ستحصل على:

### فتح النظام
- **قرطبة للتوريدات.url**
  - ينقلك مباشرة للنظام في المتصفح
  - http://localhost:5000

### إدارة النظام
- **تشغيل قرطبة.bat**
  - لتشغيل النظام
  
- **إيقاف قرطبة.bat**
  - لإيقاف النظام

- **حالة النظام.bat**
  - لفحص حالة التشغيل

- **نسخة احتياطية.bat**
  - لعمل نسخة احتياطية من البيانات

## 🆘 حل المشاكل الشائعة

### النظام لا يعمل
1. شغل "حالة النظام" من سطح المكتب
2. إذا كان متوقف، شغل "تشغيل قرطبة"
3. إذا لم يعمل، أعد تشغيل الحاسوب

### لا يمكن الوصول للنظام
1. تأكد من أن النظام يعمل
2. جرب http://localhost:5000
3. تأكد من عدم حجب جدار الحماية

### مشاكل قاعدة البيانات
1. أعد تشغيل خدمة PostgreSQL:
   ```cmd
   net stop postgresql-x64-13
   net start postgresql-x64-13
   ```
2. أعد تشغيل النظام

### بطء في الأداء
1. تأكد من توفر ذاكرة كافية
2. أغلق البرامج غير المستخدمة
3. فحص القرص الصلب
